what is IP address

IP address in unique identifier numbers that used to every device connected to the internet.

IPv4: 192.168.1.1 (most common)

IPv6: 2001:0db8:85a3:0000:0000:8a2e:0370:7334 (newer, longer version)

what is Domain?

A domain is the name of website. it's the part of the URL that identifies the which website you want visit. like, domain name: google.com

Example: https://www.google.com/search

Second-Level Domain (SLD) – e.g., google

Top-Level Domain (TLD) – e.g., .com, .org, .in, etc.

Your browser uses the Domain name, but behind the scenes, it connects using the IP address.
